# How to Update the App

## Checking for Updates

1. Open the app and go to the Dashboard.
2. Click the "Check for Updates" button. The app will check for the latest release on GitHub.
3. If an update is available, you will see a notification with the option to "Update Now."

## Updating the App

1. Click the "Update Now" button when prompted.
2. The app will download the latest version and guide you through the update process.
3. Follow any on-screen instructions. The app may restart to complete the update.

## Troubleshooting

- If the update fails, ensure you have an internet connection and try again.
- For manual updates, visit the GitHub releases page and download the latest installer.
